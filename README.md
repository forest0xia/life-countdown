# Life Hourglass | 生命沙漏

A visual life countdown timer that displays the passage of time through beautiful, interactive visualizations.

## Features

- Life progress tracking with customizable birth date and expected lifespan
- Multiple visualization modes including hourglass particles, ring charts, and desk clock
- Glass-morphism UI with ambient background animations
- Font picker and auto-scaling for desk clock display
- Mobile-friendly responsive design

## Usage

Open `life-countdown.html` in any modern browser — no build step or server required.
